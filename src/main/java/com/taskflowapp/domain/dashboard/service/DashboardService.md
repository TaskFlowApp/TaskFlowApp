# 대시보드 API 서비스

## 개요
대시보드 비즈니스 로직
 조회(read-only) 트랜잭션으로 통계 및 요약 데이터를 구성합니다.
 CRUD 관점: R(조회)에 해당하며, 쓰기/수정/삭제는 포함하지 않습니다.


## [대시보드 통계 생성]
 대시보드 통계 생성
 전체 작업 수, 상태별 수, 완료율 등을 계산하여 `DashboardStatsResponse`로 반환합니다.
 @return 집계된 통계 DTO

## [내 작업 요약 생성]
 오늘 할 일/예정/기한 초과 작업을 분류하여 `MyTasksResponse`로 반환합니다.
 @param userId 현재 사용자 식별자 (권한 컨텍스트에서 추출)
 @return 작업 요약 DTO

## [팀별 진행률]
 각 팀의 전체 작업 대비 완료(DONE) 작업 비율을 백분율로 계산합니다.
 @return 팀 이름 ➜ 진행률(%) 맵
